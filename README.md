# 📝 Todo App

A simple and responsive Todo App built using **HTML, CSS, and JavaScript**.

This app allows users to:

- Add todos
- Edit todos
- Delete todos
- Mark todos as completed
- Set priority levels (Low, Medium, High)
- Filter todos (All, Pending, Completed)
- Save data using Local Storage

---

# 🚀 Features

## ✅ Add Todo
Users can add new tasks.

Example:
- Buy groceries
- Complete project
- Learn JavaScript

---

## ✏️ Edit Todo
Users can edit:
- Todo text
- Priority level

---

## ❌ Delete Todo
Users can remove unwanted todos.

---

## ✔️ Click Todo to Complete

Instead of using checkboxs.

- User clicks on the todo text
- Todo gets line-through styling
- Clicking again removes line-through

---

## 🔥 Priority Levels

Each todo can have:

| Priority | Description |
|----------|-------------|
| Low | Small tasks |
| Medium | Important tasks |
| High | Urgent tasks |

---

## 🔍 Filter Options

The app contains 3 filters:

### 1. All Todos
Shows every todo.

### 2. Pending Todos
Shows only incomplete tasks.

### 3. Completed Todos
Shows only completed tasks.

---

## 💾 Local Storage

All todos are saved in browser localStorage.

This means:
- Data remains after page refresh
- No database required
